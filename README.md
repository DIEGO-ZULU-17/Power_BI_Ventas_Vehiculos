# Power BI - Ventas de Vehículos

Este repositorio contiene un archivo de Power BI desarrollado como parte de un proyecto práctico de análisis de ventas para un concesionario de vehículos. El trabajo está orientado a la aplicación de conceptos avanzados de DAX y a la creación de un dashboard interactivo que permite explorar métricas clave del negocio.

## Descripción del proyecto

El archivo de Power BI tiene como objetivo mostrar de forma visual y analítica el rendimiento de las ventas de vehículos, facilitando la identificación de tendencias, comparativas y oportunidades de mejora. El proyecto fue concebido como una muestra de habilidades en modelado de datos, diseño de reportes y desarrollo de medidas calculadas con DAX.

## Objetivo

- Analizar el comportamiento de las ventas de vehículos.
- Visualizar indicadores clave de negocio (ventas, volumen, rendimiento por modelo, marca o periodo).
- Aplicar técnicas avanzadas de DAX para crear métricas dinámicas y segmentaciones útiles.
- Servir como ejemplo de trabajo práctico basado en los apuntes de un curso de DAX avanzado.

## Contenido

- Archivo de Power BI (.pbix) con el dashboard principal.
- Estructura de datos preparada para análisis.
- Medidas y cálculos implementados con DAX.
- Visualizaciones interactivas orientadas a la toma de decisiones.

## Enfoque técnico

Este proyecto incorpora buenas prácticas en:

- Modelado de datos en Power BI.
- Diseño de informes y visualizaciones.
- Implementación de medidas con DAX avanzado.
- Uso de contextos de filtro y agregaciones para obtener análisis más precisos.

## Relación con los apuntes del curso

El proyecto está alineado con los conceptos abordados en el curso de DAX avanzado y complementa los apuntes contenidos en el PDF de estudio. Su finalidad es convertir los conocimientos teóricos en una solución práctica y visualmente clara.

## Requisitos

Para abrir y trabajar con el archivo de Power BI se recomienda:

- Power BI Desktop instalado.
- Conexión a los datos utilizados en el modelo.
- Conocimientos básicos de DAX y visualización de datos.

## Uso

1. Descarga o abre el archivo de Power BI en Power BI Desktop.
2. Explora las páginas del dashboard.
3. Ajusta filtros y segmentaciones para analizar diferentes escenarios.
4. Utiliza las medidas calculadas para profundizar en las métricas del negocio.

## Nota

Este proyecto puede servir como base para futuras ampliaciones, como la incorporación de nuevas dimensiones, comparativas temporales o análisis más detallados por canal, zona o vendedor.
